# What is React and why would we use it?

A JS library for building user interfaces. 

- A highly 'reactive', instantaneous library that breaks away from more traditional HTML request and response cycles. 
- JS allows manipulation in the browser, without having to send a new server request.
- React is an extra building block ontop of JS, a client side library it helps build modern, reactice UIs for websites. 
- It helps making more complex UIs easier by adoptiong a modular approach that lets us write more high level, rather than low level, code.
