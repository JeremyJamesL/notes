# React.js alternatives

React is lean & focused component based library. It doesn't have all features (e.g routing), which can be added via community packages and 3rd party libraries. 

- An alternative to React is **Angular**, which is similarly component based, but ships with more features. It can be overkill for small projects, but more suitable for larger ones.  
- Another alternative is **Vue.js**. Also component based, with more features than React, but less than Angular. It ships with routing, for instance. Not as much overload than Angular.
