# Building Single-Page-Applications (SPAs)

While we can use React to control part of HTML pages, such as a widget, it is more common to use React to control the **entire frontend** of an application. 

Once an HTML request is returned, React takes over and handles the remaining load and effectively controls the whole page. Helps dynamically change content without returning a new server request. E.g switching to 'My List' in Netflix. It may appear as if a new request is made and a new page is rendered, however, it is all handled by React.
